Welcome to the Project

This Flask application provides a platform to manage user accounts and perform search operations based on user-defined criteria.

Features include:

  • User registration and login
  • Search functionality for various parameters
  • Dashboard to view and manage search results

To get started, you can register a new account or log in if you already have one. Once logged in, you will have access to the dashboard where you can perform searches and view results.

Feel free to explore and use the features provided. If you have any questions or need help, don't hesitate to contact support.
